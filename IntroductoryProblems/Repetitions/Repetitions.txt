Description:

You are given a DNA sequence consisting of the characters A, C, G, and T. 
Your task is to find the longest repetition in the sequence, which is the maximum-length substring containing only one type of character.

Input:
A single line containing a string of n characters.

Output:
Print one integer: the length of the longest repetition.

Constraints:
1 ≤ n ≤ 10^6

Example:

Input:
ATTCGGGA

Output:
3
